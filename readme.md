Extracts marketplace data from either page searches, or a whole scan if using a mods like Auctioneer. Will save min and max prices for an item. 


```
	["Thunderstrike-EU"] = {
		["2025-05-05 03:00"] = {
			[3576] = {
				["minUnitPrice"] = 115,
				["itemName"] = "Tin Bar",
				["maxUnitPrice"] = 200,
				["itemId"] = 3576,
			},
		},
	}
```
